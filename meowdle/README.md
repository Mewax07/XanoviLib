# Meowdle

Meowdle est une **bibliothèque pour interagir avec Moodle**, créée par [LiterateInk](https://github.com/LiterateInk).

Moodle est largement utilisé dans les lycée et écoles supérieurs françaises, mais l'API fournie par Meowdle le rend plus simple d'accès et plus approfondie et utile.
