# MAQueue

[MAQueue](https://github.com/Vexcited/maqueue) est une **bibliothèque utilitaire** créée par [Vexcited](https://github.com/Vexcited).

Elle offre un ensemble d'outils qui simplifient grandement le processus de développement d'applications et de la gestion de queue asynchrone.
